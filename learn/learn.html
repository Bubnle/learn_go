分号规则

go 语言不需要分号！！！

除非一行有多条语句。

实际上编译器会将特定符号后面的换行符转换为分号。

如：

标识符，整数，浮点数，虚数，字符或字符串文字。

关键字 break, continue, fallthrough, return 等。

运算符和分隔符 ++ -- ） ] }。

eg：x + y 如果再 + 之前换行 x 标识符会触发其后面的换行符转为分号导致错误。

如果再 + 之后换行 + 不会影响后面的换行符不会转化为分号。

os 包

os 包以一种跨平台的方式，提供了一些与操作系统交互的函数和变量。

程序的命令行参数可以使用 os.Args 获取。

os.Args[0] 就是本身的名字，其他的元素是程序启动传给他的参数。

那么这些参数就是 os.Args[1:len(os.Args)] 从头到尾，也可以 [1:]。

变量声明的时候没有给值就默认是 0 值或者空字符串。

os 包

标准循环，其中的三个部分其实都可以省略

for init ; condition ; post {}

这个就相当于 while 循环

for condition {}

还有一种range的写法

for _,arg range os.Args[1:]{ 这个是省略索引 所以使用下划线 }

for i os.Args[1:]{ 这个是使用索引 其中i是当前索引也就是 1 }
